Windows 10 SMB settings

Posted on 2020-06-24 Edited on 2022-08-16

对Windows10 SMB功能的支持有限，不是很兼容，如果一定要用的话方法如下。

Windows 10系统，控制面板–>小图标–>程序和功能–>（左边）启用或关闭Windows功能–>勾选SMB 1.0/CIFS File Sharing Support (文件共享支持)。然后重启！

Windows 10同一个用户名貌似显示多个（输入用户名不是设备名），不行换一下试试。
